Multi Column Layout (Newspaper Style)

CSS Multi-column layout allows you to distribute content into multiple columns without using tables. It is useful for articles, blogs, magazines, and documentation.

The column-count decides the number of columns, while column-gap gives spacing between them. Column rules can be added using column-rule properties to show a vertical line separation.

When headings or titles should cover full width of the container, we use column-span: all. Column-width helps browser automatically adjust layout based on available screen width.
